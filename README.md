# Projeto_IoT_identificacao
## Requisitos
### Funcionais
- Cadastro usuário
- Cadastro área
- Cadastro roteador
- Cadastro tipo de usuário
- Mapeamento
- Histórico de localização do usuário
- Histórico de acesso por área
- Busca de alunos
### Não Funcionais
- Arquitetura
- Protocolo Wi-fi
- Protocolo MQTT
- Servidor
- Roteadores(min 2)
- Dispositivos conectados
